# task_angular
Manegador de tareas en angularjs

## Install
    npm install
    bower install

## Create script js y css
    gulp scripts
    gulp css
    
## Run mongodb
    mongod -dbpath=C:/mongodb/data/db
## Run server
Inside root folder execute:

    node server.js
Or

    npm start

## Optionally run with postgres database
	node server_pg.js
